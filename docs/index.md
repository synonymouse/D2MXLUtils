# Индекс кода D2Stats.au3

Этот документ описывает структуру файла `D2Stats.au3` для облегчения рефакторинга. Код разбит на логические блоки с указанием строк и кратким описанием функциональности.

| Логический блок | Строки | Описание |
| :--- | :--- | :--- |
| **Header & Configuration** | 1 - 57 | Подключение библиотек, директивы компилятора, проверка прав администратора, проверка запуска единственной копии (Singleton). |
| **Global Variables** | 58 - 159 | Объявление глобальных переменных, констант, перечислений (Enum) для цветов, флагов уведомлений и массивов кэширования. |
| **Entry Point** | 161 - 167 | Точка входа в скрипт. Регистрация функций выхода, создание GUI и запуск основного цикла `Main()`. |
| **Main Loop & Lifecycle** | 168 - 289 | Основной цикл программы (`Main`). Обработка подключения к процессу игры (`UpdateHandle`), логирование (`_Log`, `_Debug`) и корректное завершение работы (`_Exit`). |
| **Hotkeys Logic** | 291 - 416 | Реализация хоткеев: копирование статов в буфер обмена (`HotKey_CopyStatsToClipboard`), копирование информации о предметах, сравнение статов (`CompareStats`). |
| **Stat Reading Engine** | 418 - 681 | Логика чтения статов из памяти. `UpdateStatValues` обновляет кэш статов. Содержит сложную логику расчета урона оружия (`CalculateWeaponDamage`) и исправления ошибок отображения (`FixStats`), специфичных для движка D2. |
| **Drop Notifier Core** | 683 - 1411 | Самый массивный блок. Отвечает за фильтр лута. Включает парсинг правил (`NotifierCompile`), сканирование предметов на земле (`NotifierMain`), фильтрацию (`OnGroundFilterItems`) и формирование уведомлений (`DisplayNotification`). |
| **GUI Helpers** | 1413 - 1609 | Вспомогательные функции для построения интерфейса: расчет координат, создание динамических лейблов и опций (`_GUI_NewOption`, `_GUI_NewItem`). |
| **GUI Event Handlers** | 1611 - 2001 | Обработчики событий интерфейса: обновление значений (`UpdateGUI`), нажатия кнопок (`OnClick_...`), управление вкладками и скроллинг опций. |
| **Overlay System** | 2002 - 2230 | Система отрисовки прозрачного оверлея поверх окна игры. `PrintString` выводит текст (уведомления) на экран, `CleanUpExpiredText` удаляет старые сообщения. |
| **GUI Construction** | 2232 - 2501 | Функция `CreateGUI`. Создает главное окно, все вкладки (Basic, Page 1/2, Notifier, Options) и элементы управления. Содержит статические определения лейблов для статов. |
| **Settings Management** | 2503 - 2609 | Сохранение и загрузка настроек из `.ini` файла (`SaveGUISettings`, `LoadGUISettings`). Синхронизация состояния GUI с внутренними переменными. |
| **System Helpers** | 2610 - 2670 | Обработка системных сообщений Windows (`WM_COMMAND`, `WM_MOUSEWHEEL`), расчет DPI экрана и логика уведомления о гоблинах (`GoblinAlert`). |
| **Memory Injection** | 2672 - 2892 | Низкоуровневая работа с памятью. Внедрение ассемблерного кода в процесс игры (`InjectFunctions`, `InjectCode`) для вызова внутренних функций D2 (получение имени предмета, статов). Управление загрузкой DLL. |
