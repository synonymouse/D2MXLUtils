# Индекс кода D2StatsOldVersion.au3

Этот документ описывает структуру файла `D2StatsOldVerison.au3` для облегчения навигации и рефакторинга. Код разбит на логические блоки с указанием строк и кратким описанием функциональности.

| Логический блок | Строки | Описание |
| :--- | :--- | :--- |
| **Header & Configuration** | 1 - 56 | Подключение библиотек (#include), директивы компилятора (#pragma), обработка звуковых уведомлений через командную строку, проверка Singleton, прав администратора и расположения exe. |
| **Global Options & Entry Point** | 57 - 68 | Настройки AutoIt (MustDeclareVars, GUICloseOnESC, GUIOnEventMode), вызов `DefineGlobals()`, регистрация `_Exit()`, создание GUI и запуск `Main()`. |
| **Main Loop & Lifecycle** | 69 - 167 | Основной цикл программы (`Main`). Управление хоткеями, обновление хендла процесса D2 (`UpdateHandle`), проверка состояния игры (`IsIngame`), инжект функций, включение nopickup, mousefix, показ предметов, запуск нотификатора. Функции `_Exit`, `_CloseHandle`, `_Debug`, `_Log`, `_LogSave`. |
| **Hotkeys Logic** | 168 - 397 | Реализация хоткеев: копирование статов в буфер (`HotKey_CopyStatsToClipboard`), копирование списка предметов (`HotKey_CopyItemsToClipboard`), копирование текста предмета (`HotKey_CopyItem`), переключение DropFilter (`HotKey_DropFilter`), переключение показа предметов (`HotKey_ToggleShowItems`), чтение статов (`HotKey_ReadStats`), сравнение статов (`CompareStats`). |
| **Stat Reading Engine** | 398 - 631 | Логика чтения статов из памяти. `GetUnitToRead` определяет юнит (игрок/наемник). `UpdateStatValueMem` читает статы из памяти. `UpdateStatValues` обновляет кэш статов, вычисляет бонусы. `CopyStatsArray` копирует кэш для сравнения. `GetUnitWeapon` получает оружие юнита. `CalculateWeaponDamage` рассчитывает урон оружия. `FixStats` исправляет специфичные ошибки отображения D2 (скорости, регенерация, EWD по типу оружия). `GetStatValue` возвращает значение стата. |
| **Drop Notifier Core** | 632 - 1003 | Ядро системы уведомлений о луте. `NotifierFlag` и `NotifierFlagRef` работают с флагами фильтра. `NotifierCache` кэширует имена предметов и их тиры. `NotifierCompileFlag`, `GetStatsGroups`, `NotifierCompileLine`, `NotifierCompile` компилируют правила фильтра. `NotifierHelp` показывает справку с примерами. `NotifierMain` - главный цикл сканирования предметов на земле, матчинг с правилами, обработка гоблинов. |
| **Drop Notifier Processing** | 1004 - 1199 | Обработка уведомлений. `ProcessItems` формирует пул уведомлений. `DisplayItemOnGround` показывает/скрывает предмет. `OnGroundFilterItems` фильтрует предметы по правилам show/hide. `FormatNotifications` форматирует текст уведомлений (имя, тип, статы, цвет, эфирность, тир уников). `DisplayNotification` выводит уведомление. `NarrowNotificationsPool` приоритизирует уведомления (по статам > по цвету > по количеству флагов). `HighlightStats` подсвечивает совпавшие статы. `NotifierPlaySound` воспроизводит звук. |
| **GUI Helper Functions** | 1200 - 1400 | Вспомогательные функции GUI: `_GUI_StringWidth`, `_GUI_LineY`, `_GUI_GroupX`, `_GUI_GroupFirst/Next`, `_GUI_ItemCount`, `_GUI_NewItem`, `_GUI_NewText`, `_GUI_NewTextBasic`, `_GUI_ItemByRef`, `_GUI_OptionCount`, `_GUI_NewOption` (создание чекбоксов и хоткей-контролов), `_GUI_OptionByRef`, `_GUI_OptionExists`, `_GUI_OptionID`, `_GUI_OptionType`, `_GUI_Option`, `_GUI_Volume`. |
| **GUI Event Handlers** | 1401 - 1800 | Обработчики событий интерфейса: `UpdateGUI` обновляет значения статов на экране. `OnClick_ReadStats`, `OnClick_Tab`. Управление правилами нотификатора: `OnChange_NotifyRulesCombo`, `OnClick_NotifyNew/Rename/Delete/Save/Reset/Help/Default`, `OnChange_NotifyEdit`. Вспомогательные функции для файлов правил: `GetNotifierRulesName`, `GetNotifierRulesFilePath`, `SaveCurrentNotifierRulesToFile`, `CreateNotifierRulesFile`, `AskUserForNotifierRulesName`, `RefreshNotifyRulesCombo`. |
| **GUI Construction** | 1801 - 2000 | Функции создания интерфейса. `OnChange_VolumeSlider`, `OnClick_VolumeTest`, `OnClick_Forum`. `CreateGUI` - создание главного окна, вкладок (Basic, Page 1, Page 2, Options, Hotkeys, Notifier, Sounds, About), элементов управления. Статические определения лейблов для статов персонажа (уровень, опыт, золото, сигнеты, чармы, MF/GF, базовые/бонусные статы, скорости, резисты, урон и т.д.). |
| **GUI Options & Settings** | 2001 - 2200 | Продолжение CreateGUI (вкладки Options, Hotkeys, Notifier, Sounds, About). `UpdateGUIOptions` синхронизирует контролы с настройками, управляет хоткеями. `SaveGUISettings` / `LoadGUISettings` сохраняют/загружают настройки в .ini. `SaveGUIVolume` / `LoadGUIVolume` для громкости звуков. `DummySelectAll` для Ctrl+A в Edit. `WM_COMMAND` обработчик изменений в NotifyEdit. `_GetDPI` получает DPI экрана. `GoblinAlert` и `CheckGoblinHaveSeenBefore` для оповещения о гоблинах. |
| **Memory Injection** | 2201 - 2599 | Низкоуровневая работа с памятью процесса D2. `RemoteThread` создает удаленный поток. `SwapEndian` конвертирует endianness. `PrintString` выводит текст в игру через хук MedianXL. `GetItemName`, `GetItemStats`, `GetUnitStat` получают данные предмета. `GetOutputString`, `GetOutputNumber` читают результат из памяти. `WriteString`, `WriteWString` записывают строки. `GetDropFilterHandle`, `InjectDropFilter`, `EjectDropFilter` управляют DropFilter.dll. `IsMouseFixEnabled`, `ToggleMouseFix` для фикса мыши. `InjectCode` записывает ассемблерный код. `InjectFunctions` внедряет все хуки (печать, получение строк, имени/статов предмета). `UpdateDllHandles` получает адреса DLL (D2Client, D2Common, D2Win, D2Lang, D2Sigma). |
| **Global Variables** | 2600 - 2707 | Функция `DefineGlobals()`. Объявление всех глобальных переменных: константы цветов, enum'ы для печати и качества предметов, массивы GUI, флаги нотификатора, настройки звуков, кэши статов и предметов, хендлы DLL, указатели на инжектированный код, список опций GUI (nopickup, mousefix, goblin-alert, unique-tier, notify-enabled, копирование, фильтр, хоткеи), ID гоблинов. |
